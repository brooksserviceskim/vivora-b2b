# VIVORA — B2B 바이어 소개 홈페이지

한 페이지 몰입형 스크롤 · **17개 언어** · 제품 360° 회전 뷰어 · 제형 모션 · 글로벌 맵 · 인증 · 바이어 문의 폼.
순수 정적 HTML/CSS/JS (별도 서버·빌드 불필요).

## 구성
```
website/
├─ index.html                  단일 페이지 (히어로→브랜드→기술→성분→제품→Why→글로벌→인증→문의)
├─ assets/
│  ├─ css/style.css            디자인 전체
│  ├─ js/translations.js       17개 언어 번역 데이터 (문구는 여기서만 수정)
│  ├─ js/main.js               엔진: 언어전환·360뷰어·제형·차트·맵·문의폼
│  ├─ img/logo/                vivora.png, vivora-white.png, vivora-joseon.png
│  ├─ img/spin/<제품>/         360 회전 프레임 (01.png…)
│  ├─ img/texture/             제형 사진
│  ├─ img/symbol/              선비·여왕·공주·벚꽃 상징 이미지
│  ├─ img/media/               박람회 사진, 유리 홍보 영상(mp4), (제형/배경 영상 자리)
│  └─ catalog/                 브랜드 소개서·제품 카탈로그 PDF (다운로드 버튼 연결)
├─ README.md
└─ VIVORA_에셋제작_가이드.md   ← 360 컷·제형 영상 ChatGPT/영상 프롬프트
```

## 로컬 미리보기
브라우저에서 `index.html`을 그냥 열어도 동작합니다(번역을 fetch가 아닌 스크립트로 넣어 둠).
더 정확히 보려면 폴더에서:
```
python3 -m http.server 8080
```
→ 브라우저에서 `http://localhost:8080`

## 지원 언어 (17)
영어(en)·한국어(ko)·중국어(zh)·일본어(ja)·독일어(de)·프랑스어(fr)·스페인어(es)·이탈리아어(it)·포르투갈어(pt)·네덜란드어(nl)·폴란드어(pl)·러시아어(ru)·불가리아어(bg)·세르비아어(sr)·헝가리어(hu)·베트남어(vi)·태국어(th)
- 헤더 우측 🌐 버튼으로 전환. 선택값은 브라우저에 저장되어 다음 방문 때 유지됩니다.
- 첫 방문 시 브라우저 언어 자동 감지(없으면 영어).
- `?lang=ja` 처럼 URL로도 지정 가능(공유·QR용).
- **문구 수정은 `assets/js/translations.js`에서만** — 키 이름은 그대로 두고 값(따옴표 안)만 고치면 전 페이지 자동 반영.
> 작은 언어(불가리아·세르비아·헝가리·태국·베트남 등)는 출시 전 현지 원어민 감수를 한 번 받으시길 권장합니다.

## 바이어 문의 폼
제출 시 입력값이 채워진 메일이 **vivoralabs@naver.com** 으로 열립니다(mailto 방식 — 별도 서버 불필요).
- 자동 전송형(서버 없이 받은편지함 적재)을 원하시면 Formspree·Web3Forms 같은 무료 폼 백엔드로 5분 내 교체 가능합니다. 말씀 주세요.
